# skin_drug_delivery
Tools to analyze skin drug delivery

# Installation
Save the file [skin_drug_delivery.ijm](https://raw.githubusercontent.com/MontpellierRessourcesImagerie/skin_drug_delivery/refs/heads/main/skin_drug_delivery.ijm) into the folder ``macros/toolsets`` of your FIJI-installation. Select the toolset from the ``>>``-button.
