# skin_drug_delivery
Tools to analyze skin drug delivery

# Installation
Save the file [skin_drug_delivery.ijm](https://raw.githubusercontent.com/MontpellierRessourcesImagerie/skin_drug_delivery/refs/heads/main/skin_drug_delivery.ijm) into the folder ``macros/toolsets`` of your FIJI-installation. Select the toolset ``skin_drug_delivery`` from the ``>>``-button. 

<img width="260" height="34" alt="image" src="https://github.com/user-attachments/assets/fcf0ec85-9115-4fc9-99cb-97357449fdc9" />

Press the ``Install or Update Tool``-button (a cloud with a downwards arrow). The first time you do this, jython might have to be installed. In this case, accept the jython installation. After jython installation finished, close FIJI and open it again. Press the ``Install or Update Tool``-button again. A dialog showing the available versions on github will be displayed.
